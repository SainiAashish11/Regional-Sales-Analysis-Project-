# Regional-Sales-Analysis

## Overview
- In this Project, **Regional Sales Data** is studied and analysed.
- This project focuses on applying data analytics techniques to the sales domain.
- The goal is to derive insights that can help improve sales performance and drive business strategies.

## Dataset and format
- The Excel file provided in the repository is used in the project.
- Data is in Tabular format

## Files Included
- Detailed reports
- Code files
- Documentation on analytics methodologies
- Visualizations and dashboards

## Technical Skills and Tools used:
- Python - Data Cleaning and EDA
- Colab Notebook - Providing Python environment and leveraging detailed code reports and visuals
- Excel - Loading the Data into sheets
- Power BI - Dashboarding
